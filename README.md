# Get Stated

1. clone โปรเจค psuapp-announcement
```bash
git clone https://github.com/MeeMeChu/exam-printing-9.git
```
2. cd เข้าไปยังไฟล์ที่ clone มา
```bash
cd exam-printing-9
```
3. ติดตั้ง package ด้วย คำสั่ง 
```bash
npm install
```
OR
```bash
npm i
```
4. รันโปรเจค
```bash
npm run dev
```
